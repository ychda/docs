# 常见的一些问题

    ## 常用的链接
        - [Turtle](https://docs.python.org/zh-cn/3/library/turtle.html)
    
    ## 入门代码
    
    ```python
    from turtle import *
    # 这里添加代码
    done()
    ```

    或者

    ```python
    import turtle
    # 这里添加代码
    turtle.done()
    ```

    如果不添加最后一行，程序运行结束后窗口就自动关闭。
