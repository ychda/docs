## 入门代码

1. Hello, World!
  ```python
  from turtle import *
  # 这里添加代码
  done()
  ```

  或者

  ```python
  import turtle
  # 这里添加代码
  turtle.done()
  ```

  如果不添加最后一行，程序运行结束后窗口就自动关闭。
注意这两种写法的区别。**官方文档中的示例代码都是第二种，为了简单一些，可以用第一种，但是要把示例代码中的`turtle.circle(120, 180)`改为`circle(120, 180)`**。意思是把每行开头的`tuttle.`删除才行。
